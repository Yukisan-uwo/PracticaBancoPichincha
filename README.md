## 📋 Descripción del Proyecto

**Banco Pichincha** es una aplicación móvil desarrollada en Java para Android que proporciona una interfaz intuitiva y segura para que los usuarios accedan a servicios bancarios digitales. La aplicación implementa múltiples métodos de autenticación para garantizar la seguridad de las cuentas de los usuarios.

### Versión Actual: 9.5.0

## ✨ Características Principales

### 🔐 Métodos de Autenticación
La aplicación ofrece tres opciones de acceso seguro:

1. **Usuario y Contraseña** 
   - Autenticación tradicional con credenciales de MiContr
   - Validación de identidad de usuario

2. **Huella / Face ID**
   - Autenticación biométrica con huella dactilar
   - Reconocimiento facial para acceso rápido y seguro
   - Tecnología moderna para mayor comodidad

3. **PIN de 6 dígitos**
   - Método alternativo de acceso rápido
   - Código numérico personalizado para mayor seguridad

### 🌐 Servicios Disponibles

- **Ubicanos** 📍 - Localiza las sucursales más cercanas
- **Clave digital** 🔑 - Gestiona tus claves de acceso
- **Llámanos** ☎️ - Contacto directo con soporte bancario

### 👥 Acceso Multi-usuario
- Opción de ingresar con otro usuario
- Manejo seguro de múltiples cuentas
- Cambio rápido entre usuarios
